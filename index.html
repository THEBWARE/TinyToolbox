TinyToolbox

The Ultimate Python Utility Pack

Text Utilities

Convert text to uppercase, lowercase, title case, leetspeak, Morse code, reverse text, count words/characters, and generate random text.

File & System Utilities

Rename multiple files, find and delete duplicates, get system info (RAM, CPU, OS details), search files, calculate file size, and count file types.

Internet Tools

Ping websites, get IP address & geolocation info, shorten/expand URLs, download files, and check weather.

Security & Encryption

Generate random passwords, encrypt/decrypt text (AES, Base64), hash files (SHA256, MD5), generate SSH keys, check password strength, and encrypt/decrypt files.

Conversion Tools

Convert images (JPG ↔ PNG ↔ WEBP), units (length, weight, temperature, currency), numbers (binary ↔ decimal ↔ hex), timezones, and text to speech.

Productivity & Misc

Manage to-do lists, generate QR codes, roll dice, flip coins, countdown timer, stopwatch, random number generator, and Base64 encode/decode.
